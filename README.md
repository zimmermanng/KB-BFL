# Knowledge Badge "Barrierefrei lehren"

Dieses öffentliche Repository dient als zentrale Dateiablage für das Knowledge Badge "Barrierefrei lehren". Hier befinden sich alle relevanten Unterlagen, Materialien und Ressourcen zum Projekt.

**Hinweis:** Nicht öffentliche und vertrauliche Unterlagen (bspw. die Prüfungsfragen) sind nicht in diesem GitHub-Projekt enthalten, sondern an einem separaten, geschützten Ablageort hinterlegt.

## Wiki

Weitere Informationen zu Syllabi, Prüfungsterminen, Merkblättern etc. finden Sie im [KB-BFL Wiki](https://github.com/zimmermanng/KB-BFL/wiki)

## Ticketübersicht

Die Übersicht aller Projekt-Issues ist über den folgenden Link oder den Menüpunkt "Issues" erreichbar: [KB-BFL Issues](https://github.com/zimmermanng/KB-BFL/issues)

Die Issues können über Labels wie bspw. "Orgateam" oder "Prüfungsrunde SS2026" gefiltert werden.

Für die Prüfungsrunde im Frühjahr 2026 wurde ein eigener Milestone mit allen zugehörigen Issues angelegt: [KB-BFL Prüfung Frühjahr 2026](https://github.com/zimmermanng/KB-BFL/milestone/2).
